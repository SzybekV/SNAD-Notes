- [[Primitive Types]]
	- Small Enough to be put in on the Stack.
	- In JavaScript it is a Fundamental data type in JS that represents simple atomic values.
		- `let name = "nish"` String
		- `let num = 42` Int
		- `let num = 4.2` Float
		- `let isTrue = true` Bool
		- `let n = null` Null
		- `let undefined` Undefinded
